Sobre o verificador de contraste

Esta ferramenta foi criada para que designers e desenvolvedores possam testar a conformidade do contraste de cores com base nas Diretrizes de acessibilidade de conteúdo da Web (WCAG), conforme estabelecido pelo World Wide Web Consortium (W3C).Esses cálculos são baseados nas fórmulas especificadas pelo W3C.


Como usar?

Para executar seu teste, selecione uma cor de primeiro plano e uma cor de fundo. Os resultados serão gerados automaticamente e você receberá suas pontuações, podendo saber se cada critério foi ou não aprovado. Você pode selecionar 'Adicionar no histórico' para adicionar sua amostra a uma lista gerada abaixo.
